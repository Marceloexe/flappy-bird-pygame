## 📝 Sobre o Projeto
Este é um clone do clássico jogo Flappy Bird, desenvolvido em **Python** utilizando a biblioteca **Pygame**. 

O projeto foi construído durante um curso prático realizado aos sábados no SENAI. O objetivo principal foi colocar em prática conceitos fundamentais de desenvolvimento, como manipulação de eventos, renderização gráfica 2D e lógica de programação orientada a objetos.

## 🚀 Tecnologias Utilizadas
* **Python** 
* **Pygame**
